# Anforderungen

## Software
- **AutoHotkey**: Version 1.1.37.02
- **PowerShell**: Version 5.1 oder höher

## Bibliotheken
- JSON.ahk: JSON-Unterstützung für AutoHotkey

## Konfiguration
- `apikey.xml`: Verschlüsselte Datei mit API-Key
- `prompts.json`: JSON-Datei mit vordefinierten Prompts
